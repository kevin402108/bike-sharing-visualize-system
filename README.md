# 自行车共享系统

自行车共享系统是传统自行车租赁的新一代形式，从会员注册、租赁到归还的整个过程都实现了自动化。通过这些系统，用户可以从特定位置轻松租用自行车并在另一个位置归还。目前，全球约有500多个自行车共享项目，拥有超过50万辆自行车。如今，由于自行车共享系统在交通、环境和健康问题方面的重要作用，它们受到了广泛关注。

除了自行车共享系统有趣的实际应用外，这些系统生成的数据特性也使其对研究具有吸引力。与其他交通服务（如公共汽车或地铁）相比，自行车共享系统明确记录了行程持续时间、出发位置和到达位置。这一特性使自行车共享系统成为一个虚拟传感器网络，可用于感知城市中的移动性。因此，通过监测这些数据，可以检测到城市中大多数重要事件。

## 属性信息
hour.csv 和 day.csv 都包含以下字段，除了 hr（小时）字段在 day.csv 中不可用

- **instant**: 记录索引
- **dteday**: 日期
- **season**: 季节 (1:春季, 2:夏季, 3:秋季, 4:冬季)
- **yr**: 年份 (0: 2011, 1:2012)
- **mnth**: 月份 (1 到 12)
- **hr**: 小时 (0 到 23)
- **holiday**: 该日是否为假日
- **weekday**: 星期几
- **workingday**: 如果该日既不是周末也不是假日则为1，否则为0
- **weathersit**: 天气状况
  - 1: 晴朗，少云，部分多云
  - 2: 薄雾+多云，薄雾+破碎云，薄雾+少云
  - 3: 小雪，小雨+雷暴+散云
  - 4: 大雨+冰雹+雷暴+薄雾，雪+雾
- **temp**: 标准化摄氏温度。值通过(t-t_min)/(t_max-t_min)计算得出，t_min=-8，t_max=+39（仅在小时尺度上）
- **atemp**: 标准化体感温度（摄氏）。值通过(t-t_min)/(t_max-t_min)计算得出，t_min=-16，t_max=+50（仅在小时尺度上）
- **hum**: 标准化湿度。值除以100（最大值）
- **windspeed**: 标准化风速。值除以67（最大值）
- **casual**: 临时用户数量
- **registered**: 注册用户数量
- **cnt**: 总租赁自行车数量，包括临时用户和注册用户

如需更多信息，请访问以下链接：
http://archive.ics.uci.edu/ml/datasets/Bike+Sharing+Dataset
